# Пошаговая настройка Notion и подключение к Cursor через MCP

В этом документе описан процесс настройки базы данных в Notion с нуля и её интеграции с Cursor. Мы не используем готовые шаблоны, а собираем свою систему.

## Этап 1: Создание аккаунта и подготовка Notion

1. **Регистрация в Notion:**
   - Перейдите на сайт [notion.so](https://www.notion.so/).
   - Нажмите **"Get Notion free"**.
   - Зарегистрируйтесь с помощью Google-аккаунта или email.
   - При вопросе о цели использования выберите **"For personal use"** (Для личного использования) — это полностью бесплатно.

2. **Очистка рабочего пространства (Опционально):**
   - Если у вас уже есть старые записи (как на скриншоте "Заметки"), **не удаляйте их**, если они вам нужны.
   - Стандартные шаблоны (Roadmap, Meeting Notes, Docs, To-Do), которые вам не нужны, можно удалить: наведите на них курсор в левом меню, нажмите на три точки (`...`) и выберите **"Delete"**.

3. **Создание главной страницы проекта:**
   - В левом меню нажмите **"+ Add a page"** (или "New page").
   - В появившемся окне выберите **"New 'Cheshbon Hanefesh' page"** (самый первый вариант с иконкой пустого листа).
   - *Критически важно: Не выбирайте "Table" или "Database" на этом этапе! Иначе вся страница превратится в одну таблицу, и вы не сможете добавить туда другие базы данных. Нам нужна именно пустая страница-контейнер.*
   - Выберите иконку (например, 🎲 или ⚙️) и обложку (Add cover) для настроения.

## Этап 2: Создание баз данных (Архитектура Life OS)

Согласно нашей архитектуре, нам нужно создать 5 связанных баз данных. Начнем с создания структуры на главной странице.

1. **Создание базы "Сферы жизни" (Areas):**
   - На странице `Cheshbon Hanefesh` наберите `/table` и выберите **"Table view"** (в разделе Database).
   - В появившемся меню справа выберите **"New empty data source"**.
   - **Как назвать таблицу:** Наведите курсор на слово `Table` (в сером овале над колонками) -> нажмите правую кнопку мыши (или кликните на три точки рядом) -> выберите `Rename` -> впишите `Сферы жизни`.
   - **Колонка "Название":** Самая первая колонка (с иконкой `Aa`) — это всегда название. Кликните на её заголовок и переименуйте в `Название сферы`.
   - **Колонка "Статус сферы":** Нажмите на серый плюсик `+` (справа от колонок) -> выберите тип `Status`. Переименуйте колонку в `Статус`. 
     - Нажмите на эту колонку -> `Edit property`.
     - Настройте статусы: `В планах` (To-do), `Активно` (In progress), `Архив` (Done).

2. **Создание базы "Проекты" (Projects):**
   - **Важно:** Не нажимайте на серую надпись `+ New page` прямо под таблицей (она добавляет новую строку в текущую таблицу).
   - Кликните мышкой на пустое белое место **сильно ниже** таблицы (например, отступив пару сантиметров вниз).
   - Если курсор не появляется, наведите мышку под таблицу, пока не появится серый плюсик `+` слева, и нажмите на него.
   - Снова наберите `/table` -> выберите **"Table view"**.
   - В меню справа выберите **"New empty data source"**.
   - Переименуйте её в `Проекты` (кликнув правой кнопкой на слово Table).
   - Колонки: `Название проекта` (переименуйте первую колонку Aa).
   - **Связь со Сферами:** Нажмите `+` -> выберите тип **Relation** -> выберите базу данных `Сферы жизни` -> нажмите **Add relation**.

3. **Создание базы "Задачи и Привычки" (Tasks & Habits):**
   - Снова `/table` -> **"Table view"** -> **"New empty data source"**.
   - Назовите её `Задачи и Привычки`.
   - Колонки: 
     - `Name` (Название)
     - `Статус` (Status: To-do, In progress, Done)
     - `Тип` (Select: Разовая задача, Привычка)
     - `Статус привычки` (Select: Активно, На автомате)
     - `Срывы` (Number)
     - `Проект` (Relation -> выберите базу "Проекты")
     - `Сфера` (Relation -> выберите базу "Сферы жизни")
     - `Блокирует / Заблокировано` (Позже настроим через Dependencies)

*(Остальные базы и сложные формулы мы настроим чуть позже, когда свяжем Notion с Cursor, чтобы Cursor мог помочь с формулами).*

## Этап 3: Получение API-токена Notion для Cursor

Чтобы Cursor мог управлять вашей базой данных, нужно создать интеграцию (бота) в Notion.

1. **Создание интеграции:**
   - Перейдите по ссылке: [notion.so/my-integrations](https://www.notion.so/my-integrations).
   - Нажмите кнопку **"New integration"**.
   - Выберите ваше рабочее пространство (Workspace).
   - Введите имя (например, `Cursor MCP Bot`).
   - Нажмите **"Submit"**.

2. **Сохранение токена:**
   - На следующей странице вы увидите **"Internal Integration Secret"** (длинный набор символов, начинающийся с `secret_...`).
   - Нажмите **"Copy"**. **Сохраните этот токен в надежном месте** (он понадобится нам на следующем этапе).

3. **Настройка прав доступа (Capabilities):**
   - В меню слева (в настройках интеграции) перейдите в раздел **"Capabilities"**.
   - Убедитесь, что стоят галочки:
     - Read content
     - Update content
     - Insert content
   - Нажмите **"Save changes"**.

## Этап 4: Подключение интеграции к вашей базе данных

Бот создан, но пока он не видит вашу страницу `Cheshbon Hanefesh`. Ему нужно дать доступ.

1. Вернитесь в свой Notion на страницу `Cheshbon Hanefesh`.
2. В правом верхнем углу страницы нажмите на три точки (`...`).
3. Прокрутите вниз и выберите **"Connect to"** (или "Connections" -> "Connect to").
4. В поиске введите имя вашей интеграции (`Cursor MCP Bot`).
5. Выберите её и нажмите **"Confirm"**.
   *Теперь бот имеет доступ к этой странице и всем базам данных внутри неё.*

## Этап 5: Подключение MCP сервера в Cursor

Теперь, когда у вас есть токен (Internal Integration Secret), мы можем подключить Notion к Cursor.

1. Откройте настройки Cursor (нажмите на шестеренку в правом верхнем углу или `Ctrl + Shift + J`, затем `Settings`).
2. В левом меню найдите раздел **Features** и прокрутите вниз до **MCP** (Model Context Protocol).
3. Нажмите кнопку **`+ Add New MCP Server`**.
4. Заполните поля следующим образом:
   - **Name:** `notion`
   - **Type:** `command`
   - **Command:** `npx -y @notionhq/notion-mcp-server`
5. Раскройте секцию **Environment Variables** (если она скрыта) и добавьте переменную:
   - **Key:** `NOTION_TOKEN`
   - **Value:** *вставьте сюда ваш скопированный токен (начинается на secret_... или ntn_...)*
6. Нажмите **Save** (или просто закройте окно, настройки сохранятся).
7. Рядом с добавленным сервером `notion` должен загореться зеленый кружок (статус "Connected").

*Примечание: Для работы этой команды у вас на компьютере должен быть установлен Node.js (если его нет, Cursor может выдать ошибку, тогда нужно будет скачать его с nodejs.org).*
